# BULLY-Scholarship-Edition-Easy-Install
This is for the people who are horribly confused by shady links.

# #1: Install .iso file
The .iso contains the game, Most important step.
To install, Mount the .iso by right clicking on it and clicking Mount. If you do not see mount then right click and select "Open with.." Then choose "WINDOWS EXPLORER".

# #2: Install patch (use if you are on windows 7 and above)
Download the patch .EXE in the "Patch" folder of this repo, MAKE SURE TO NOT SKIP!
The patch .EXE has to be installed after setup.exe, No exceptions.
To install the patch, You need to add it into the bully game folder (not the iso)
Bully should be found at:
```
C:\Program Files (x86)\Rockstar Games\Bully Scholarship Edition
```
Run the patch installer after placing it in the root of the directory for Bully.

# Now you are done with the installation of Bully: Scholarship Edition!
